# Student Performance Dashboard

An interactive Student Performance Dashboard built using Python and Streamlit.

## Features

- Login Authentication
- Dark/Light Theme
- Interactive Plotly Charts
- Student Search
- Rank System
- CGPA Calculator
- Attendance Tracking
- CSV Upload
- Pass/Fail Analysis

## Technologies Used

- Python
- Pandas
- Plotly
- Streamlit

## Run Locally

```bash
streamlit run app.py
